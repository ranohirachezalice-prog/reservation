⚙️ Étape unique de configuration

Avant la première utilisation, il faut renseigner votre Client ID Google dans le fichier HTML.

Comment faire :
1. Ouvrez console.cloud.google.com
2. Créez un projet → API & Services → Identifiants
3. Créez un ID client OAuth 2.0 (Application Web)
4. Dans "Origines JS autorisées" : ajoutez l'URL où vous hébergez ce fichier (ou http://localhost pour tester)
5. Activez l'API Google Sheets dans votre projet
6. Copiez votre Client ID et remplacez dans ce fichier :
